Handwritten Digit Recognition Interface

This application allows you to explore and experiment with handwritten digit classification using deep learning models trained on the MNIST dataset.

What You Can Do:

  • Select existing models (e.g., fully connected or convolutional neural networks) from the left menu to test their predictions.
  • Draw a digit (0–9) using your mouse on the canvas, or upload an image for recognition.
  • Create and train your own models using the + Add Model option — stack layers like Dense, Conv2D, MaxPooling, etc., and save them for testing.

About the Dataset:

The MNIST dataset contains 70,000 grayscale images of handwritten digits (28x28 pixels), split into:

  • 60,000 training images
  • 10,000 testing images

Predefined Models:

  • Model 1 (model1): A simple fully connected neural network with two hidden layers of 128 neurons each.
    Architecture: Flatten → Dense(128) → Dense(128) → Dense(10)
    Test Accuracy: 97.69%   |   Loss: 0.0780
  • Model 2 (model2): A convolutional neural network (CNN) with two convolutional layers and max pooling.
    Architecture: Conv2D(32) → MaxPool → Conv2D(64) → MaxPool → Flatten → Dense(128) → Dense(10)
    Test Accuracy: 98.95%   |   Loss: 0.0354

Start by choosing a model from the sidebar or click + Add Model to build and train your own custom architecture!
